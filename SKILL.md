# Skill: 文件内容分析器 (file_analyzer)

## 基本信息

| 属性 | 值 |
|------|-----|
| **名称** | file_analyzer |
| **版本** | 1.0.0 |
| **描述** | 分析文件内容，提取统计信息、关键词和代码结构 |
| **作者** | DevOps Team |
| **语言** | 中文/English |

---

## 功能特性

- **统计模式**: 行数、字符数、空行、注释行统计
- **关键词模式**: 高频词汇和关键标识符提取
- **结构模式**: 函数、类、导入语句识别
- **完整模式**: 以上所有分析组合

---

## 参数定义

### 输入参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `file_path` | string | ✅ | - | 要分析的本地文件绝对路径 |
| `analysis_type` | enum | ❌ | `full` | 分析类型：`stats`/`keywords`/`structure`/`full` |

### 参数示例

```json
{
  "file_path": "/mnt/kimi/upload/main.py",
  "analysis_type": "full"
}
